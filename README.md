# Sistema de Gestión de e-commerce en Go

Este proyecto corresponde a un sistema básico de gestión de e-commerce desarrollado en Golang. Permite registrar productos, crear clientes, generar pedidos y calcular subtotal, IVA y total de una compra.

## Módulos
- Productos
- Clientes
- Pedidos
- Cálculo de compra
- Resumen de pedido

## Estructura
```txt
sistema-ecommerce-go/
├── main.go
├── go.mod
├── README.md
├── .gitignore
├── models/
│   └── models.go
└── services/
    ├── productos.go
    ├── clientes.go
    └── pedidos.go
```

## Ejecución
```bash
go run .
```
